## Code

### Arguments

- `collapsed` useful for longer or redundant code that still needs to be accessible
- `span-[1-6]` defines the width of the code specimen

### Examples


Simple example
```code
codeception
```

Simple example markup
```code|collapsed
'''code
codeception
'''
```

Simple example markup markup
```code
'''code|collapsed
  '''code
  codeception
  '''
'''
```

## Button

### Keys
- __`url: string`__ defines the link destination
- `title: string` provides an optional description (uses the url if omitted)

### Arguments
- `span-[1-6]` defines the button width

### Example

```button
  {"url":"/#/button", "title":"Button documentation"}
```
